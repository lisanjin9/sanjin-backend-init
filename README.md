# 创建一个万用模板
[个人博客地址✈](lisanjin9.top)
## 基础设置
````
# 项目名称
spring:
  application:
    name: sanjin-backend-init
# 端口
server:
  port: 8080
  
# 数据库配置
datasource:
  driver-class-name: com.mysql.cj.jdbc.Driver
  url: jdbc:mysql://localhost:3306/my_db
  username: root
  password: root
````
